## P0 ‚Äî Critical

### P0-1 ‚Äî Copy pipeline client suggestions are completely invisible to admin
- **Area:** Admin‚ÜíClient Handoff
- **Files:** `server/copy-review.ts`, `src/components/client/ClientCopyReview.tsx`
- **Issue:** When a client submits a "Suggest Changes" edit, it writes to `copy_sections.client_suggestions`. No admin component reads or renders this column anywhere.
